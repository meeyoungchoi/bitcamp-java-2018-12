POST 요청 - 파일 업로드(multipart/form-data)

첨부 파일의 데이터를 서버에 보내려면 "multipart/form-data" 형식으로 전송해야 한다. form태그의 encotype 속성을 "mulipart/form-data"로 변경하라

현재 html의 url을 기준으로 요청 url을 상대적으로 기술할 수 있다.
